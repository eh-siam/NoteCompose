NoteCompose is a modern note-taking Android application built using Jetpack Compose. It provides a simple, fast, and clean UI for creating, editing, and managing personal notes efficiently.

The app is designed with a focus on smooth user experience and minimal UI, allowing users to quickly capture ideas, tasks, and important information anytime.

🚀 Features:
Create, update, and delete notes easily
Clean and modern Jetpack Compose UI
Fast and lightweight performance
Local data storage for offline access
Simple and user-friendly interface
🛠️ Tech Stack:
Kotlin
Jetpack Compose
Android Architecture Components
